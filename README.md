# Monolith 🌑

**Monolith** is a massive, self-contained 64-bit Virtual CPU Architecture, Compiler, and OS ecosystem.

## Features
- **Custom VM**: Fully simulated CPU with 64 general-purpose registers and memory mapping.
- **AetherScript**: A custom high-level programming language compiler (lexer, AST parser, optimizer).
- **Assembler**: Translates custom assembly mnemonics directly into Aether bytecode.
- **Mini-OS**: Bootloader and kernel simulation written entirely in Aether Assembly.
